# Markdown

Github를 사용하기 위한 마크다운 문법 정리

---

# 1. 제목

## 1-1. 줄(호라이즌) 같이 쓰는 방법

- 사용법

```
html h1 size and hr
===

html h2 size and hr
---
```
- 적용

html h1 size and hr
===

html h2 size and hr
---

- 정리

> h1 사이즈 + 호라이즌: 문장 밑에 '==='

> h2 사이즈 + 호라이즌: 문장 밑에 '---'

----

## 1-2. 크기만 조절하는 방법

- 사용법
```
# this is h1
## this is h2
### this is h3
#### this is h4
##### this is h5
###### this is h6
```

- 적용

# this is h1
## this is h2
### this is h3
#### this is h4
##### this is h5
###### this is h6

> 제일 큰 사이즈는 h1, 제일 작은 사이즈는 h6

> 문장 앞에 '#'을 사이즈만큼 붙인다. ex) h4는 #을 4번 붙이면 된다.
