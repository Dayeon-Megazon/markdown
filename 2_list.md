# 2. 리스트 사용하기
## 2-1. 기호로 표시하기

- 사용법
```
* 1
  * 1-1
    * 1-1-1

+ 2
  + 2-1
    + 2-1-1

- 3
  - 3-1
    - 3-1-1
```

- 적용

* 1
  * 1-1
    * 1-1-1

+ 2
  + 2-1
    + 2-1-1

- 3
  - 3-1
    - 3-1-1

> [TAB] 키를 사용하지 않으면, 아래와 같이 나온다.

*example*
- 사용법
```
* 4
* 4-1
* 4-1-1
```
- 적용
* 4
* 4-1
* 4-1-1
